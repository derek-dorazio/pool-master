# Archived Legacy Web App

This directory contains the retired pre-PoolMaster React web app.

Rules for agents and developers:

- Do not treat this app as an active implementation target.
- Do not fix builds, tests, contracts, or CI by editing files here.
- Do not use this app as the source of truth for backend contracts or current product behavior.
- Planning agents may inspect it only for historical use cases, layout ideas, and component inspiration.
- Implementation work must happen in `clients/poolmaster`.

This directory is intentionally excluded from the active build, test, coverage, and deployment flow.
