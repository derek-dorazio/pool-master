# Glossary

- `Provider-imported event`: a real-world sporting event discovered from a data
  provider and imported into PoolMaster
- `Event field`: the participant set for a specific event
- `Contest field`: the frozen contest-specific interpretation of the event
  field that entries use
- `Seeded template`: a product-defined reusable contest configuration shipped
  with the system rather than managed through normal UI flows
- `Live scoring`: automated backend propagation of provider updates into entry
  scores and leaderboard order
- `Contest history`: the league-facing archive of completed contests
- `Exceptional admin action`: operational intervention for unusual problems, not
  a normal user workflow
